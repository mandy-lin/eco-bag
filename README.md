# eco-bag
此項目為大三課程作業，由小組共同製作
負責的部分前端為主html、css，架構的討論以及文件的製作。
在限塑的政策下，我們有了租用各種購物袋的想法

系統主要功能:
1.不塑之客(會員系統):
分為3種身分來選擇，分別為訪客，會員和管理者。
訪客：擁有基本的功能，像是瀏覽網頁，查詢和了解最新資訊
會員:除了有訪客的功能外，另外還可以到討論區留言，還有租借的功能。
管理者: 擁有所有權限，可以新增修改資料。
會員申請
會員登入
忘記密碼
基本資料查詢及修改
修改密碼
2.塑站速決(典藏系統):
收藏每個地點有提供哪些種類的塑膠袋，也介紹每種塑膠袋有甚麼功用。
3.查詢系統:
查詢那個地點是否有自己要用或剩餘的塑膠袋，免得消費者到了定點後沒有自己想用的塑膠袋的困擾。
4.備份與還原:
運用SQL Server系統所提供的備分與還原機制定期進行備分與還原。
5.討論區/留言板:
可以在網路上和網友們討論說哪幾種的塑膠袋實用性高，或者可以留言說哪個地方比較需要哪種種類的塑膠袋或那裡有問題等等。
6.塑塑離去(秘密武器):
提供用戶回收不必要的塑膠袋，如果提供到一定的數量就免費租借一次。
